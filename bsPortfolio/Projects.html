All Projects

DOORGUY April 2020

This was a project for my senior AP Computer Science class in high school and I earned college credit through Boise State. My teacher was Bob Williamson and this was at Skyview High School in Nampa, ID. This was made with MIT App Inventor.

GLOAMAugust 2022

GLOAM is a website for a fake inflatables business. The process became an exercise of utalizing JavaScript in various ways such as timers, accordions, and some JavaScript plug-ins. JQuery is also used here.


MIMIROctober 2022

This is a concept of a Destiny 2 Companion App. There are concept maps, a written proposal, a PowerPoint, a YouTube video presenting the PowerPoint, and an early prototype.

Jiggy CentroNovember 2022

Utalizing Vue 3.0 JavaScript framework, users can use this website to select their favorite fighters from Super Smash Bros Ultimate. Uses vue-router.

Spiral Shoes & Co.November 2022

One of my first exercises using Bootstrap. Features JQuery date countdown, JQuery Navigation Drop-down animation, and a good example of a static, single-page E-Commerce page.

Shaymin.ggDecember 2022

Using React, a JavaScript Framework, Shaymin.gg pulls information from Pokemon GO's API to display the base attack, defense, and stamina stats for all availble pokemon.

Cut-to-Static RecordsMay 2023

This was an exercise in learning SQL databases and PHP. It's the backend website for maintaining a record store's inventory.

Top-Down Wilderness SurvivalAugust 2023

Made in Unreal Engine for Fortnite. Features a camera that constantly updates and tracks the player. The objective is to fight wolves and find parts to repair your car.

PEPPERMINTAugust 2023

Made in Unreal Engine for Fortnite. A 2D-Platformer, 3D-Shooter, 3D-Melee, story-driven experience.

Effigy GoodsOctober 2023

A solo project where I made an E-Commerce website with SQL and PHPmyadmin. Complete with a login feature.

WWSapsscOctober 2023

Working with Western Waterworks Supply & Fabrication, I created a program that automates some of the processes when adding pipe in AutoCAD. Please watch the video and read the description.

Flood EscapeNovember 2023

Made in Unreal Engine for Fortnite. My first mini-game like experience. Features an in-game shop, and opt-in/out game system, multiple maps, and cool gameplay.

SHATTERED: A 3D RPGDecember 2023

Made with Unity, I made a 3D RPG akin to the Persona Series or Pokemon. Features include persisting data across scenes, turn-based instanced fights, cutscenes, dialog, and more.

Scott's Furniture BarnDecember 2023

A collaborative assignment in my SWDV280 class, myself and six others were assigned to create a website for our client, Scott. This is an E-Commerce page complete with a LAMP stack and is an exercise for our frontend and backend development.
